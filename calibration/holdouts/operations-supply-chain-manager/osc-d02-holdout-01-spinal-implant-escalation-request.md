# Value Analysis Product Request

**Request Date**: 2026-03-11
**Requestor**: Talia Fen, MBA, RN, Service Line Administrator, Neurosciences
**Product Requested**: ApexRise Expandable TLIF Cage, Helix Ortho Systems, Catalog AR-TLIF-32
**Current Product**: ArcSpan PEEK TLIF Cage, Meridian Spine Technologies, Catalog ASP-TLIF-28

## Clinical Rationale
- The spine service requested the expandable cage for selected lumbar fusion cases where the operating surgeon believes in-situ expansion may improve fit and restore disc height with less trialing.
- The request did not present evidence supporting broad replacement of the current formulary cage across routine degenerative cases.
- The current formulary device remains clinically acceptable for the large majority of TLIF volume based on existing service-line use and prior committee review.
- Implant traceability and lot-level documentation must remain equivalent to the current workflow before any restricted clinical use begins.

## Committee Decision
- Decision: Deferred pending restricted-use protocol and competitive pricing counteroffer
- Condition 1: Spine surgeons must define indication criteria for use in complex deformity or revision cases rather than open access across all lumbar fusion volume.
- Condition 2: Supply chain will seek a target price aligned to current category guardrails before the request returns to committee.
- Condition 3: OR leadership must validate point-of-use UDI capture performance for the requested packaging format before any trial begins.
- Condition 4: If the item returns for approval, initial use will be capped to a named surgeon cohort with 90-day utilization review.

**Decision Date**: 2026-03-26
**Approved by**: Nolan Quist, MD, Chair, Surgical Value Analysis Subcommittee

## Implementation Notes
- No item master activation will occur until pricing, indication criteria, and traceability controls are documented.
- The spine service line will submit case-selection criteria and a measurable outcomes review plan before reconsideration.
- Supply chain finance will model rebate exposure if restricted use exceeds the proposed cap.
- If approved later, consignment terms must include lot visibility and same-day implant log reconciliation.
